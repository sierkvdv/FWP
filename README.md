## 🚀 Features

- Modern React + TypeScript setup
- Responsive design met Tailwind CSS
- Smooth animations met Framer Motion
- Project showcase met filtering
- Contact formulier
- SEO optimized

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Build Tool**: Create React App

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/sierkvdv/FWP.git

# Navigate to project directory
cd FWP

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🌐 Deployment

This project is configured for deployment on Vercel. The build process has been optimized to resolve TypeScript import issues.

## 📁 Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── data/          # Static data (projects, skills, contact)
├── utils/         # Utility functions and constants
├── types/         # TypeScript type definitions
└── assets/        # Images and other static assets
```

## 🎨 Design System

The website uses a dark theme with:
- **Primary Color**: Cyan accent (#00D4FF)
- **Background**: Dark gradients
- **Typography**: Modern sans-serif fonts
- **Animations**: Smooth transitions and micro-interactions

## 📱 Responsive Design

Fully responsive design that works on:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)
